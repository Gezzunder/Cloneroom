This mission demonstrates using Maca's M3Editor to produce a snazzy replacement debug box for A3 Epoch.

load the mission and hit preview.  Once in the mission, you are in a stock debug room.
Mousescroll to select delete and it's removed, select spawn to see the demo amphitheatre.
Use delete to clear once more and then select editor.  Load the amphi project and have a play.

amphi.sqf is the M3Editor project for the amphitheatre
amphi_sqf.sqf is the output from M3Editor using the amphi project
